About the System

System technical specifications and academic project abstract

Academic Abstract

The Skill Gap Analysis Agent is an AI-powered educational technology tool designed to identify competency disparities in students and job seekers. Transitioning from academic studies to industry roles requires specific domain capabilities that are often missing from general curricula.

By parsing uploaded PDF resumes, the system extracts core competencies. It compares these findings against targeted job criteria (drawn from 14 professional fields). The mismatch findings generate a personalized readiness percentage, an 8-week structured roadmap, certification paths, and recommended course materials. A built-in AI career advisor assists users through real-time chat dialogues.

Technology Stack

Backend Architecture: Python, Flask, SQLite database
Frontend Design: Bootstrap 5, Vanilla CSS3, Javascript, Chart.js
Artificial Intelligence: Google Gemini 1.5 Flash API
Resume Parsing: pdfplumber, PyPDF2, spaCy NLP
Core Objectives
  • Extract resume skills using combined NLP and keyword algorithms.
  • Map client capabilities against predefined required role skills.
  • Generate readiness scores and interactive match charts.
  • Structure week-by-week 8-week learning roadmaps.
  • Recommend top courses, certifications, and up-skilling projects.
